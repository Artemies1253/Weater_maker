# Weater_maker
Стек технологий: Python 3.9, Pony, OpenCV, requests, BeautifulSoup, регулярные выражения.
Программа получения и выведения прогноза погоды с сайта mail.ru.
Модуль движок с консольным интерфейсом, способный.
1.Обновлять базу за запрашиваемый период
2.Выводить на консоль с базы данных погоду за запрашиваемый период
3.Рисовать открытку с базы данных погоду за запрашиваемый период сроком, выводит максимум 1 неделю
Открытка рисуется с помощью OpenCV, в зависимости от типа погоды меняется иконка и градиент:
солнечно - от желтого к белому, дождь - от синего к белому, снег - от голубого к белому, облачно - от серого к белому
При старте автоматически загружыется в базу данных информация за следующию неделю.
Программа работает только по 2021 году.
Для запуска дрона достаточно запустить модуль user_interface.py.
